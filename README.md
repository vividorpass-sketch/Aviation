# Aviation
In this project, we talk about ways of improving communication and fast transportation. In the recent times aviation has been the centre of communication with different contries and continents. Here it is broken down to a simpler understanding to all ages.
A modern, interactive aviation website with a complete flight booking system. This project demonstrates a professional airline booking platform with three interconnected HTML page, a styling feature CSS and a functional Javascript.

# overview
SkyWings Airlines is a comprehensive flight booking website that showcases the art of modern aviation. The project includes:
- **Homepage** - Introduction to aviation and airline services
- **About Page** - Company information
- **Flight Search Page** - Booking you flight
